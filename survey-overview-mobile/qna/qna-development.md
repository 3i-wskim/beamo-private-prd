# QnA — 개발 (Development)

> 개발 관련 자유 질의응답. PRD에 직접 영향을 미치지 않는 수시 소통 채널.
> 유의미한 주제는 ADR로 승격될 수 있음.
>
> **룰**: `해결 여부`는 **질문자 본인만** 변경할 수 있음. 답변이 달려도 질문자가 납득해야 해결 처리.

### Q. 코멘트/스레드 수정 API 403 Forbidden

| 항목 | 내용 |
|------|------|
| **작성자** | wooseung.kim |
| **작성일** | 2026-06-05 |
| **해결 여부** | ✅ 해결 |
| **질문** | `POST /v2/com/comments/{id}` 엔드포인트로 스레드 답글 텍스트 수정 시 403 Forbidden 응답. |

#### A.

| 항목 | 내용 |
|------|------|
| **답변자** | wooseung.kim |
| **답변일** | 2026-06-05 |
| **답변** | 서버 API 스펙 확인 결과 **PUT** 메서드 사용 (`PUT /v2/com/comments/{id}`). 클라이언트가 `@POST`로 호출하고 있어서 403 발생. `@PUT`으로 수정 완료. |
