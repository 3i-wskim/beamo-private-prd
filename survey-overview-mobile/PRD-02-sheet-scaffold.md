# PRD-02: 바텀 시트 스캐폴드 📐

> 기획 스펙: SO-04 (Bottom Feature Bar + Content Sheet Scaffold)

---

## 한줄 요약

> 하단 4탭 피처 바 + min/mid/full 3단계 콘텐츠 시트. 드래그·스크롤 연동으로 리사이즈하며, PRD-03~05의 모든 콘텐츠가 이 스캐폴드 위에서 동작.

**기존 활용**: `BeamoDraggableBottomSheet`, `NestedScrollConnection`, per-tab `LazyListState` 관리 동작 중.

---

## 🔲 바텀 피처 바

**4개 탭** (고정 순서): `Tags` → `Comments` → `Box` → `Checklist`

- 화면 하단 고정 (시스템 내비게이션 바 위)
- 각 탭: 아이콘 + 레이블 · 활성 탭 색상 강조
- ⚡ **네트워크 상태 무관 항상 표시**

### 탭 전환 동작

| 상황 | 액션 | 결과 |
|------|------|------|
| 시트 닫힘 | 아무 탭 탭 | 해당 탭 콘텐츠로 **mid** 크기 열기 |
| 시트 열림 | **같은 탭** 다시 탭 | 리스트 루트로 리셋 + 스크롤 상단 + **mid** |
| 시트 열림 | **다른 탭** 탭 | 콘텐츠 전환 + 루트 뷰 + **현재 크기 유지** |

---

## 📏 시트 크기

| 크기 | 높이 | 보이는 것 |
|------|------|----------|
| **Min** | 헤더만 | 제목 행만 · 콘텐츠 숨김 · Tags/Comments: 레이블+➕ / Box/Checklist: 레이블만 (현행 코드 기준, Figma 디자인 우선) |
| **Mid** | ~50% | 리스트 상반부 · 캔버스 상반부 |
| **Full** | 상단 70% (피처 바 포함) | 거의 전체 · 캔버스+내비 컨트롤은 위에 |

> 📌 시트는 항상 **mid로 열림** · 탭 전환 시 크기 유지 · Full에서 FAB 자연 가림 (의도적)

### 리스트 vs 디테일 레이아웃

| 뷰 타입 | 피처 바 관계 |
|---------|-------------|
| **리스트** (루트 뷰) | 피처 바 **위에서 시작** — 오버레이 안 함 |
| **디테일** 뷰 | 피처 바를 **오버레이** |

- 시트 위 캔버스 영역 → 패닝/줌 인터랙션 유지
- 시트 배경: 불투명 · 상단 둥근 모서리

---

## ↕️ 드래거

- 시트 상단 **pill 형태** 드래그 핸들
- 터치 타겟: 핸들 위아래 최소 ~44px
- 드래그 → 연속 위치 변경 → 릴리즈 시 **가장 가까운 크기로 스냅**
  - 스냅 임계값: 화면 높이 ±3% 또는 ±30px (먼저 도달)
  - 겹치는 영역: 마지막 이동 방향 기준 판단
  - 애니메이션: spring 또는 ease-out

> 📌 리스트 스크롤 위치 유지 — min에서 다시 열면 복원

---

## 🔄 Scroll-to-Resize

| 스크롤 방향 | 조건 | 결과 |
|------------|------|------|
| ⬇️ 아래 | 아래 콘텐츠 있음 | 정상 리스트 스크롤 |
| ⬇️ 아래 | 리스트 최상단 (`scrollPos=0`) | 시트 한 단계 축소 (Full→Mid→Min→dismissed) |
| ⬆️ 위 | 시트 < full | 시트 **full**로 확장 |
| ⬆️ 위 | 이미 full | 정상 리스트 스크롤 (위로) |

**기존**: `NestedScrollConnection`으로 내부 LazyColumn ↔ 시트 리사이즈 연동 완료.

---

## 🧠 탭별 상태 메모리

- 각 탭: 스크롤 위치 + 내비게이션 depth 독립 기억
- **같은 탭 재탭**: 디테일 닫기 + 루트 뷰 리셋 + 스크롤 상단 + mid 리사이즈
- **다른 탭 전환**: 콘텐츠 전환, 이전 탭 상태(스크롤·depth) 보존 — 복귀 시 복원
- **시트 닫힐 때 전체 리셋** — 재열기 시 모든 탭 루트 뷰 + 스크롤 상단

**기존**: per-tab `LazyListState`, 같은 탭 재탭 → 디테일 닫기 + 스크롤 탑 + NORMAL 리사이즈 완료.
